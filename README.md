## Installation

Install Blood-Donation with npm

```bash
  git clone https://github.com/kinga-kus/Blood-Donation
  cd Blood-Donation
  npm i
  npm run start
```

Clone the project

```bash
  git clone https://github.com/kinga-kus/Blood-Donation
```

Go to the project directory

```bash
  cd Blood-Donation
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
